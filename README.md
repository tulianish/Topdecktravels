# Topdeck Travels
![Logo][logo]

A cloud-travel application built as part of CSCI 5409 - Advanced Cloud Computing.

### Technical Stack :

* Node
* Express
* AJAX
* MongoDB

### Clients :

* Website
* Mobile Application

### Containerisation :

* Docker : [Docker Image](https://hub.docker.com/repository/docker/tulianish/topdecktravels )

### Hosting :

* AWS

### AWS Services :

* AWS EC2
* Auto Scaling and Load Balancing
* SNS
* S3
